# TypeScriptプロジェクト例
できるだけ実用的かつ小さい構成を目指したTypeScriptプロジェクト  
問題点があれば教えていただけると助かります。  
   
<br>
<br>

# 各種コマンド
初期設定
```
npm install
```
<br>


実行
```
npm start
```
<br>


外部ライブラリのライセンス更新
```
npm run license
```
<br>


ビルド
```
npm run build
```
<br>

   

<br>
<br>

# 開発環境
* Windows 10 Home
* Node.js v16.15.0
* npm 8.5.5
